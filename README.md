# Sorting_Visualizer

This project is a Web Visualization tool for sorting algorithms.
I have implemented a total of 6 algorithms so far. And will try to add more later.
Algorithms implemented yet so far are


1. Selection sort
2. Insertion sort
3. Heap sort
4. Merge sort
5. Quick sort
6. Bubble Sort
